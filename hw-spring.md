## HW-3

```
У вас есть файл в формате CSV с данными о продажах продуктов в вашем магазине. 
Файл содержит информацию о продажах за каждый день в течение нескольких лет. Каждая строка файла содержит следующую информацию: 
дата продажи, наименование продукта, количество проданных единиц и общая выручка от продаж.

Ваша задача - проанализировать эти данные с помощью библиотеки pandas и построить график, отображающий общую выручку за каждый год.

Посчитайте суммарную выручку за каждый год с помощью функции pandas.sum().
Постройте график, отображающий общую выручку за каждый год, используя библиотеку matplotlib.
```
